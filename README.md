# Plex-Console
** A small handheld console with a Charlieplexed LED matrix and even a speaker! **
The way it works is simple: Have the main board, attach it to whatever output device is needed for the game you want to play, and bam! You have a small pocket-sized game console with endless possibilities of emulation. All you have to do is make your attachment board, code the game, and snap on the main board!
